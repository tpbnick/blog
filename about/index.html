About Me

Random Projects

Soccerfront.io

Soccerfront.io is a soccer live score web application that I built using Nodejs, Expressjs, HTML, CSS, and JavaScript. Soccerfront pulls data from the API-FOOTBALL API and displays the information in a easy to use live score table and league table (standings) view. This is an ongoing project and new features will be added sporadically.
